Diese Animationen und Spiele sind jetzt auch sehr einfach an zu schauen/zu spielen auf
        iPad, smartphone und tablet

mit dem gratis "Puffin Web Browser" für Apple iOS und Android.

Kopieren Sie die URL dieser Website in die Adresszeile Ihres Browsers,
laden Sie den Puffin-Webbrowser aus dem AppStore / PlayStore herunter.
Kehren Sie dann im Puffin-Webbrowser zu dieser Site zurück
durch Einfügen der kopierten URL in die Adresszeile.

Für die Quiz: Wählen Sie im Menü den Theatermodus
und aktivieren Sie den Mauszeiger (das Menü jetzt unten rechts).
Ziehen Sie den Zeiger auf das Objekt, das Sie ziehen möchten.
Heben Sie Ihren Finger an, legen Sie ihn zurück und halten Sie ihn gedrückt, bis ein blauer Kreis erscheint.
Jetzt können Sie das Objekt ziehen.
